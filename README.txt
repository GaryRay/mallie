# Building

## Requirements

* GCC (If you want OpenMP support)
* SDL2.0
* premake4

### MacOSX

Edit scripts/setup_macosx.sh, then,

  $ ./scripts/setup_macosx.sh
  $ make
 
### Linux

Edit scripts/setup_linux.sh, then,

  $ ./scripts/setup_linux.sh
  $ make

### Windows

Edit vcbuild.bat, then,

  > vcbuid.bat

Solition file will be generated.

EoL.
